```
http://127.0.0.1:3030
```

### ネットワークアクセス（同じネットワーク内の他のデバイスから）
```
http://192.168.3.92:3030
```

---

## 📱 Microsoft Edge で開く手順

### 方法1: URLを直接入力

1. **Microsoft Edge を起動**
2. アドレスバーに以下を入力:
   ```
   http://192.168.3.92:3030
   ```
3. Enter キーを押す

### 方法2: ショートカット作成

1. 上記URLにアクセス
2. Edge メニュー（…）→「アプリ」→「このサイトをアプリとしてインストール」
3. デスクトップにショートカットが作成されます

### 方法3: お気に入りに追加

1. URLにアクセス
2. `Ctrl + D` でお気に入りに追加
3. 名前を「MangaAnime配信システム」に変更
